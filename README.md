# React project template without create-react-app

### dependencies

- react, react-dom
- webpack, webpack-cli, webpack-dev-server
- babel-loader
- html-webpack-plugin
- css-loader
- @babel/core, @babel/preset-env, @babel/preset-react

### use

clone this repository and run:

```sh
npm install
```

then

```sh
npm run dev
```
